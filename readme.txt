Alice is a Multi-Channel customer service platform for your e-commerce store or online business that centralises all customer interactions and helps to manage and automate customer support.


== Description ==


Alice is a Virtual Assistant with AI and automation abilities which you can customize to create a smooth, easy to manage customer service experience that increases sales and drives costs down for your e-commerce or online business.


Like the bird which represents our brand, we stand for freedom. Once you set up and customize her, Your Alice will allow you to centralize your multi-channel customer conversations into one platform, automate most of your customer interactions with human-like AI-driven conversations, centralize, collect & analyse valuable data so you can make smarter, faster customer-driven decisions to increase your sales!


Alice centralises all of your customer facing channels into one single inbox, helping you to design automation and understand the data to make better decisions.



== Installation ==


Install Alice Chatbot as a regular WordPress plugin. Here is different ways to install Alice Chatbot plugin :

= Install by Search Plugin =
- In your Admin, go to menu Plugins > Add New
- Search Alice Chatbot
- Click to install
- Activate the plugin
- For the settings page go to WooCommerce > Settings > Alice Chatbot
- Insert the plugin auth key from Alice website

= Install by Upload Plugin =
- Download the latest version of Alice Chatbot (.zip file)
- In your Admin, go to menu Plugins > Add New
- Select  "Upload Plugin"
- Click on “Choose File”
- Select downloaded alice.zip & click on ‘Install Now’ button
- Activate the plugin
- For the settings page go to WooCommerce > Settings > Alice Chatbot
- Insert the plugin auth key from Alice website

= Where do I get the auth key? =
